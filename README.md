# perritos

Proyecto web de datos curiosos sobre perros creado con React, Vite y Tailwind CSS.

## Cómo usar

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev`
- Construir el proyecto: `npm run build`

## Despliegue

Este proyecto está preparado para desplegarse en Netlify con `dist` como carpeta de publicación.
